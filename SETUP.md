# Participant Setup

Complete these steps before the training session.

## Required access

- Access to ChatGPT
- Ability to upload a Markdown file into ChatGPT
- Ability to create or use a ChatGPT Project, if your workspace supports Projects

## Files to have open

- `Participant_Worksheet.docx`
- `templates/BRD_Template_20_Min_Demo.md`
- `prompts/chatgpt-prompts.md`
- `exercises/stale-quote-illustration-warning.md`

## Suggested ChatGPT setup

1. Start a new chat or create a new Project.
2. Upload `templates/BRD_Template_20_Min_Demo.md`.
3. Paste the opening prompt from `prompts/chatgpt-prompts.md`.
4. Answer ChatGPT's questions as if you are the business stakeholder.
5. Capture decisions, assumptions, and open questions in the worksheet.

## Data caution

Use only the training scenario. Do not enter confidential customer, policyholder, production system, or internal business data.
